## Description

This is a Quiz App.

### Features

- SignIn
- SignUp
- Different Categories for Quiz
- Dynamic Questions at each time of attempting quiz.
- Leaderboard
- Dark & Light Mode

## Tech Stack

- React
- Express JS
- Mongoose
- MongoDB
